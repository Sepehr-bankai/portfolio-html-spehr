01 / Commerce automation

Telegram Digikala Bot

FastAPI PostgreSQL Docker

A Telegram shop bot with an admin dashboard and automated Digikala price tracking. It keeps products just below marketplace pricing while protecting buy-price margins through approval flows.

  • Admin dashboard for products, orders, deposits, tickets, and broadcasts.
  • Minute-based Digikala API checks with Rial to Toman conversion.
  • Below-buy-price approval from the dashboard or Telegram admin buttons.

02 / Trading systems

Bybit CHOCH Trader

Async IO Bybit API Pandas

A cryptocurrency trading bot that scans multiple timeframes for CHOCH and MTR patterns, then places position-aware market orders with configurable TP and SL logic.

  • Priority scan from weekly down to 5 minute candles.
  • Server time sync to avoid Bybit authentication drift errors.
  • Structured JSON logs for signals, errors, and debugging.

03 / Public data

Maps Gov GE Cadastral Collector

Python JSON API Automation

A coordinate-driven workflow for collecting cadastral information from Georgia's public map service faster than manual map clicking.

  • Generates coordinate sets from latitude, longitude, and radius inputs.
  • Sends controlled POST requests to the map API.
  • Saves each coordinate response as structured JSON for review.
